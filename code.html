In the tutorial section we will show you how easy it is to create state-of-the-art Seldonian algorithms. However, these tutorials are intended as an introduction to Seldonian algorithms, do not include the backend necessary for sophisticated interfaces, and will remain simple as improved algorithms are created. We hope to release a software library that is frequently updated to contain state-of-the-art Seldonian algorithm implementations. When this library is available (hopefully January) it will be presented here.
